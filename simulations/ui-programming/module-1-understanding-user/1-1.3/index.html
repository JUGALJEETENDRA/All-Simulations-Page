📖 Learning Recap: Activity 1 (Mental Model Hunt)
What Usability Problems Were Found? Save links in unexpected headers, unlabelled hexagon indexes, missing location paths, offset alignment boxes, and vague labels.
Which Principle Was Learned? Mentals models (interfaces must correspond to standard layout conventions).
How Can It Be Applied in Real Projects? Place controls where users anticipate them, align forms on grids, and add text details to icon panels.

What Usability Problems Were Found? Abstract symbols (bananas, hourglasses) clashing with standard interface meanings.

How Can It Be Applied in Real Projects? Restrict layouts to standardized visual symbols (refresh arrows, back arrows, shields).
