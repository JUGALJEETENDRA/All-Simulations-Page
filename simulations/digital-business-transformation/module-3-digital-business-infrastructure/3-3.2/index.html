Dave Chaffey's Reference Board

Concise e-business networking infrastructure definitions

Ref
Internet: Public global network connecting clients, maximizing customer discovery.
Intranet: Private internal network restricted to corporate employees, securing records.
Extranet: Restricted portal extending intranet databases to authorized suppliers.
DNS: Distributed directory service translating human URLs into machine IPs.
HTTP & HTTPS: HTTP sends clear text; HTTPS mandates TLS encryption to secure buyer cash flows.
TCP/IP: Protocol suite formatting data into packets and guaranteeing error-free transport.
Cloud Hosting: Remote virtualized server instances replacing physical local hardware.
Edge CDN: Global edge node network serving cached static web pages locally, cutting latency.
Client-Server: Browsers (clients) request and present pages, while backend servers run database ledgers.
