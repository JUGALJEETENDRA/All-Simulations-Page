Database APIs Glossary
cursor.execute(query, params)

Binds variables options dynamically, compiles parameters values, and triggers SQL parsing on remote databases.

connection.commit()

Commits active transactions states. MySQL handles connections in sessions where writes are uncommitted until committed.

cursor.fetchall()

Extracts buffer results rows from query sets and returns standard Python tuple list objects.

cursor.close() & conn.close()

Frees server socket connection buffers and network handles allocated for client streams.
