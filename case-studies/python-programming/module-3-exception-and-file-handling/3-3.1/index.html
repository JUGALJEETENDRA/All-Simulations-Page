🕵 Detective's Diagnostic Manual

Welcome to active diagnostic operations, Detective. Master these core Python concepts to repair the network:

1. The Error Categories

  • Syntax Errors: Violations in Python grammar (missing colons, uneven brackets, wrong indentation). Detected *before* code executes.
  • Runtime Errors (Exceptions): Crashes during program run (division by zero, missing array elements, incompatible variable casts).
  • Logical Errors: Code runs fine without tracebacks, but yields mathematically incorrect or flawed results due to logical developer errors.

2. Exception Traps

  • try: Surrounds code which could potentially throw an exception.
  • except: Captures and handles the exception. Does not execute if try succeeds.
  • else: Runs *only* if the try block executes completely without exceptions.
  • finally: Runs *always*, regardless of whether an exception was raised, handled, or completely bypassed. Used for cleanup.

3. Assertion Enforcements

assert condition, message validates parameters. If condition is False, it immediately raises an AssertionError crash holding your custom details.
