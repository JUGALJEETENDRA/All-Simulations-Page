📖 Learning Recap: Activity 1 (Visual Chaos)
What Usability Problems Were Found? Clashing typefaces, off-grid inputs, weight imbalances, overlaying card locks, and notification floods.
Which Principle Was Learned? Visual hierarchy scans and typographic consistency constraints.
How Can It Be Applied in Real Projects? Build interface frameworks on strict grids, lock copy choices to single font weights, and filter alert priorities.

📖 Learning Recap: Activity 2 (Cognitive Load)
What Usability Problems Were Found? Choice paralysis caused by displaying 16 raw features at once.
Which Principle Was Learned? Miller's Law (reducing cognitive load via category chunking).
How Can It Be Applied in Real Projects? Group portal functions into clear directories to speed up selection rates.
