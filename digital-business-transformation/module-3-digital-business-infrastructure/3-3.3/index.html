Concise e-business cloud computing definitions

Ref
Cloud Computing: On-demand delivery of virtualized servers, database ledgers, and networks over public or private lines.
Virtualization: Software hypervisor layer dividing physical server nodes into isolated VM client instances.
IaaS: Raw virtualized servers, storage, and networking. You manage core OS runtimes and SQL databases.
PaaS: Managed server runtime containers. You deploy app code; vendors control OS patches and capacity.
SaaS: Turnkey business software consumed on subscription models directly inside client web browsers.
Public Cloud: Shared virtualization compute pools hosted by third-party vendors and shared globally.
Private Cloud: Dedicated server arrays hosted exclusively for one enterprise to maximize security barriers.
Hybrid Cloud: Configuration uniting public cloud auto-scaling nodes with private on-premise silos.
Multi-cloud: Utilizing separate public cloud providers to prevent server vendor locking.
